## 利用する方へ

- **ミッション読み込み画面でブラックアウトする場合はミッションファイルの書き換えに失敗しています。やり直してください。**
- **編集するファイルは念の為バックアップを必ず取りましょう。**

1. DCSの設定を日本語にする
   1.1. `[DCS保存場所]/Config/lang.cfg`ファイルの内容を`JP`に書き換える（ファイルがなければ作成する）
2. ミッション一覧の日本語化
   2.1. 翻訳したいキャンペーンファイル(**.cmp)を翻訳した`**.cmp`ファイルで置換する。
3. 各ミッションの日本語化
   3.1. 翻訳したいミッションファイル(**.miz)をzipアーカイバーで開き、`/l10n/`に翻訳した`JP`フォルダを挿入する
